This web application shows the National Heritage Sites around the world and gives details of each location. To enable different layers, use the control layer on the top right of the application to turn on different heritage sites. To use the Search by country function, uncheck all checkboxes except the Search checkbox. I used Leaflet, HTML and CSS to create this simple web map




This map visualizes New York City's Green Taxi pick up points. Green Taxis are those which run outside of the borough of Manhattan. Please note that this project calls on a huge chunk of data hence it is slower when using the timeline function. I added a timeline to show the location and time of where the pickup points were in the boroughs outside of Manhattan by the hour. There are cluster layers which will change depending on the zoom level as well.




Above is a map I created on ArcGIS Online. I gathered the shapefile data from GMU's GIS data library, cleaned up the shapefiles and added additional building polygons that were not on the original shapefiles. I put it all together on an online map for everyone to view. This web map is intended to be a interactive map for students and other users to see the buildings, parking lots and athletic fields around campus.
